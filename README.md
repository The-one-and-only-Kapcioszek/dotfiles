# dotfiles


<p>My personal dotfiles</p>
<br>

<b>wm</b>: bspwm
<br>
<b>compositor</b>: <a href="https://github.com/jonaburg/picom">picom</a>
<br>
<b>status bar</b>: <a href="https://github.com/polybar/polybar">polybar</a>
<br>
<b>terminal</b>: kitty
